Lesson - 2 Deletion
d - delete
d$ - to delete till end of line


2.3 OPERATION AND MOTIONS
	d 	motion

	w - until the start of the next word,
	e - to the end of the current word,
	$ - to the end of the line

2.4 UNDO

	u - to undo previous actions
	U - to undo all the changes on a line
	CTRL-R - To undo the undo's 

3.1 PUT

	dd - delter the line and store it in a vim register
	p  - to put the line that has been deleted.

3.2 REPLACE COMMAND

	r - replace letter

3.3 Change operator on replacing command

	ce - cut the letters from a word

3.4 MORE CHANGE USING C

	c - change operator is used the the same motions as delete.
	c [number] motion
		motion:- w(word) and $(end of line)

4.1 CURSOR LOCATION AND FILE STATUS

	CTRL-G - to show your location in the file and the file status.
	G	- to move to a line in the file.

	G      - to move to the bottom of the file
	gg     - to move to the start of the file
		type number and G to move to certain line on file

4.2 THE SEARCH COMMAND

	/   - followed by a phrase to search for the phrase
		- to search for the same pharse press 'n'
		- to search for a phrase in the backward direction, use ? instead of /.
	CTRL-O - to go back to where you come from 
	CTRL-I - to go forward

4.3 MATCHING PRANTHESES SEARCH

	%   - to find a matching ),], or }

4.4 SUBSTITUTE COMMAND
	
	:s/old/new/  - to substute 'new' for 'old', in particular line
	:s/old/new/g - adding g means globally/everywhere

	:#,#s/old/new/g - #,# are the line number of the range of line where subtution is done
	:%s/old/new/g	- to change every occurance in the while file
	:s/old/new/gc	- to find every occurrence in the whole fine, with a prompt whether to 
			  substitute or not.

5.1 EXECUTE AN EXTERNAL COMMAND

	:! - followed by an external command to execute that command
	
5.2 MORE ON WRITING FILES

	:w - to save the changes made to the text
	
5.3 SELECTING TEXT TO WRITE

	** To save part of the file, type 'v' motion :w FILENAME **
	
	v - to select the lines (like shift+arrow_key)
